Our Insurance Claim Process

  1. Free storm inspection – We check shingles, metal, flashing, vents, chimney, ridge, and soft metals for hail impact & wind lift/crease. We also look for collateral hits (gutters, window screens, fascia). Drone imaging available for steep/complex roofs.
  2. Photo & video documentation – Date-stamped images showing slope, test square hits, and mechanical creases (not foot scuffs).
  3. Adjuster coordination – We meet the adjuster on-site to review slopes, collateral damage, and code items so nothing critical is missed.
  4. Xactimate estimate – Line-item scope in the same software most carriers use. If the initial carrier scope is short, we submit supplements with documentation.
  5. Build to code & manufacturer spec – Proper underlayment, starter, hip/ridge, ventilation, flashing, and valley metal. We install systems (e.g., GAF) to qualify for enhanced warranties where appropriate.
  6. Final invoice & depreciation release – We provide completion photos and paperwork so the carrier releases recoverable depreciation (RCV policies).

What We Look For

  • Asphalt shingles: hail impact (granule loss with mat bruise), wind-creased tabs, broken seal strips, missing caps, exposed fasteners, torn valleys, compromised pipe boots, counter/step flashing failures.
  • Metal: hail dings on pans/ribs, finish micro-fractures, deformed ridge/valley/edge metals, fastener back-out on classic rib, seam integrity on standing seam.
  • Flat roofs (PVC/TPO/EPDM): membrane punctures, heat-weld/adhesive failures, scupper/termination issues, trapped moisture, hail “spidering.”
  • Collateral: gutters/downspouts, window screens, garage doors, A/C fins, soft metals—helps corroborate hail size and direction.

Insurance Basics (Quick & Plain English)

  • ACV vs RCV: Actual Cash Value pays today’s value (depreciated). Replacement Cost Value pays ACV now, then releases recoverable depreciation after completion. Policies vary.
  • Deductible: It’s your legal responsibility. We do not “waive” deductibles. That’s insurance fraud and can void claims.
  • Supplements: If the carrier scope misses code items (e.g., drip edge, valley metal, ice & water in valleys, ventilation upgrades), we submit documentation and code citations so the job meets local code and manufacturer install requirements.
  • Code upgrades: Some policies include Ordinance & Law coverage—this can pay for required upgrades (e.g., deck re-nailing, ridge vent, synthetic underlayment) up to limits.
  • Timelines: Most carriers require prompt notice after the storm. Filing delays can risk denial—call us and we’ll help you decide if a claim is warranted before you file.

FAQs

Will a small number of hail hits really matter?

If hits breach the mat or dislodge protective granules, leaks and premature aging follow. We document by slope; some slopes pass, some fail.

My neighbor got a roof—why was mine denied?

Different orientation, pitch, age, or materials. Also, adjusters and carriers differ. Solid documentation and a reasonable re-inspection request can help.

Can you meet my adjuster?

Yes. We prefer to—so the scope reflects what’s actually on your roof.
